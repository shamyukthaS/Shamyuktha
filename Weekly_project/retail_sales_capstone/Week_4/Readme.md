# Week 4 - Azure Databricks

## Project

Retail Sales Dashboard

## Objective

Implement the Retail Sales Dashboard using Azure Databricks with Bronze, Silver and Gold layers.

## Tools Used

- Azure Databricks
- Apache Spark
- Delta Tables
- SQL

## Input Files

- sales.csv
- products.csv

## Bronze Layer

Created Delta tables for:

- bronze_sales
- bronze_products

## Silver Layer

- Joined sales and product datasets
- Calculated revenue
- Calculated cost
- Calculated profit
- Saved transformed data as a Delta table

## Gold Layer

Created reports for:

- Product Summary
- Store Summary
- Top Selling Products
- Underperforming Products

## SQL Tasks

- Verified Bronze tables
- Verified Silver table
- Verified Gold tables

## Output Files

- bronze_sales.csv
- bronze_products.csv
- silver_sales_products.csv
- gold_product_summary.csv
- gold_store_summary.csv
- gold_top_products.csv
- gold_underperforming_products.csv
