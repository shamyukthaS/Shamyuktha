# Week 1 - SQL & MongoDB

## Project
Employee Attendance Tracker

## Objective

The objective of Week 1 is to design the database structure for an Employee Attendance Tracking System using MySQL and MongoDB. This includes creating relational tables, performing CRUD operations, implementing a stored procedure, and storing unstructured employee feedback using MongoDB.

---

## Technologies Used

- MySQL Workbench
- MongoDB Compass
- MongoDB Shell

- tasks

- Created Tasks table
- Inserted sample records

- Created Stored Procedure to calculate employee work hours

### MongoDB
